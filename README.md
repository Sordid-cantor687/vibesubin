## 🛠️ How to Use

vibesubin integrates into your workflow to save you time. Once installed, you can trigger the assistant inside your code environment.

### Typing the Command
To trigger the tool, type `/vibesubin` in your text editor or command console. The assistant scans the active project folder. It identifies code that no longer serves a purpose and suggests improvements.

### Security Checks
The software runs an automated security audit. It scans for hardcoded secrets such as passwords or private keys. If it detects a potential leak, it alerts you immediately and offers a fix. This keeps your credentials out of your public code history.

### Code Cleanup
Refactoring code manually takes hours. vibesubin automates the removal of dead code. The software identifies functions and variables you no longer use and flags them for deletion. This reduces the size of your project and makes it easier to read.

## 🔒 Security Features

We designed vibesubin with privacy as a priority. The tool performs all analysis locally on your machine whenever possible. It does not upload your private source code to external servers without your permission.

* **Secret Detection:** Monitors your files for accidental inclusions of keys and tokens.
* **Integrity Audits:** Compares your current codebase against best practices to find hidden vulnerabilities.
* **Version Control Cleanup:** Ensures your project history remains free of sensitive data.

**Does this software modify my files without asking?**
No. It presents a list of changes for you to review. You must approve each change before the software applies it.

**Is my data stored on the cloud?**
Your project files remain on your local machine. We do not store your code in our database.

**Can I undo changes?**
Yes. You can use your standard undo command or revert changes using your version control software.
